This is a boring dummy homepage. You get the Frank fun by doing

darcs get http://personal.cis.strath.ac.uk/~conor/pub/Frank
and the sample code file is test.fk. To run it, just compile Frank and feed the executable the file
ghc --make -O2 Main -o frank
./frank < test.fk

Oh, I forgot to mention... SHE is essential to compile Frank.
